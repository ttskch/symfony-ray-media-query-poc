## Installation

```shell
$ git clone git@github.com:ttskch/symfony-ray-media-query-poc.git
$ cd symfony-ray-media-query-poc
$ composer install
$ bin/console doctrine:migrations:migrate -n
```

## Usage

```shell
$ symfony serve --no-tls -d
$ open http://localhost:8000 
```
